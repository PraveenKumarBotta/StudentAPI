# StudentAPI (.NET + MySQL)

## Project Overview
This is a RESTful API built using ASP.NET Core and MySQL database.  
It supports CRUD operations for managing student data.

---

## Features
- Create, Read, Update, Delete (CRUD)
- Entity Framework Core with MySQL
- Swagger UI for API testing
- Clean project structure

---

## 🛠️ Technologies Used
- ASP.NET Core Web API
- Entity Framework Core
- MySQL (Pomelo Provider)
- Swagger (OpenAPI)
